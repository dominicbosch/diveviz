diveviz
=======

A platform to allow graphical representation of data and navigation through different hierarchy levels.


INSTALLATION
------------

### Prerequisites:

- PostgreSQL 9.3
	- Create user `diveviz_view_all`
	- Create database `diveviz_example`

- Node.js v0.10.28
